# ESP32 小时钟项目

## 项目环境
- **开发工具**: CLion + PlatformIO
- **设备**: 
  - ESP32 WROOM-32E
  - DHT11 传感器
  - OLED 0.96 显示屏
  - 按钮

## 页面布局
- **0页**: 显示环境温湿度等信息
- **1页**: 显示时间
- **2页**: 显示当天天气情况
- **3页**: 显示明天天气概况
- **4页**: 显示后天天气概况

## Web 页面功能
- **WiFi 配置**: 设置设备连接的 WiFi 网络
- **轮播设置**: 配置页面的轮播时间和顺序
- **实时查看环境温湿度**: 可以通过 Web 页面实时查看当前环境温湿度数据
